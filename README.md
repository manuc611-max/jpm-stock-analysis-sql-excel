# jpm-stock-analysis-sql-excel
Stock analysis using SQL and Excel (trend, volatility, moving averages)

## Overview
This project analyzes JPMorgan stock data to identify trends, volatility patterns, and overall market behavior.

## Tools Used
- SQL (data extraction & aggregation)
- Excel (analysis & visualization)

## Key Analysis
- Monthly price trends
- Price movement (Open vs Close)
- Volatility (price range)
- High vs Low comparison
- Moving average (trend smoothing)

## Key Insights
- Stock shows overall upward trend but with high volatility
- Growth is inconsistent with frequent short-term declines
- Volatility does not directly correlate with price direction
- Recent data suggests weakening momentum and possible consolidation
